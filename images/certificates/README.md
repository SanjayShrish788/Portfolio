Save your certificate images in this folder with the following exact names:

8. **hp-power-lab.jpg** - HP Power Lab 2.0 Certificate
9. **quiz-whiz.jpg** - Quiz Whiz Competition Certificate

### Image Format Guidelines:

- **Supported formats**: .jpg, .jpeg, .png, .webp
- **Recommended size**: 1200px width (height will auto-adjust)
- **File size**: Keep under 500KB for faster loading
- **Quality**: Use high-quality scans or screenshots

### How to Save Images:

1. Download/save your certificate images from the chat or your files
2. Rename them according to the list above
3. Place them in this `images/certificates/` folder
4. Refresh your portfolio website to see them displayed

### Features:

- ✅ Click on any certificate image to view it in full-screen modal
- ✅ Hover effects with "Click to view" badge
- ✅ Professional styling with shadows and animations
- ✅ Responsive design for all screen sizes
- ✅ Lazy loading for better performance

### Note:

If you use different file extensions (e.g., .png instead of .jpg), update the file paths in `js/data.js` accordingly.
